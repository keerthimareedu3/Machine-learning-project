# 🏦 Loan Approval Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a loan application will be **Approved** or **Rejected** using Machine Learning techniques. The model analyzes customer demographic and financial information to assist banks in making faster and more accurate loan approval decisions.

---

## 🎯 Objective

The objective of this project is to build a classification model that predicts loan approval status based on applicant information such as income, education, credit history, loan amount, and property area.

---

## 📊 Dataset Features

- Gender
- Married
- Dependents
- Education
- Self_Employed
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Loan_Amount_Term
- Credit_History
- Property_Area
- Loan_Status (Target Variable)

---

## 📊 Business Insights

- Customers with a good credit history have a higher chance of loan approval.
- Applicant income significantly impacts loan approval decisions.
- Loan amount and repayment term influence approval probability.
- Property area and education also contribute to prediction performance.

---

## 💡 Business Benefits

- Reduces manual loan screening time.
- Supports faster and more consistent loan approval decisions.
- Helps identify eligible applicants efficiently.
